# nth-child

# CSS nth-child Task

## What is this task?

A CSS practice task focused on using the `nth-child()` selector to select and style specific elements based on their position inside a parent element.

## File

  Contains all exercises with their CSS styling.

## Exercises

- **Exercise 1: Zig-Zag Boxes**  
  Created a group of boxes and used `nth-child()` to style even boxes differently and create a zig-zag layout.

- **Exercise 2: Hide Even Numbers**  
  Created boxes containing numbers from 1 to 10 and used `display: none` with `nth-child()` to hide the even-numbered boxes.

- **Exercise 3: Star and X Boxes**  
  Created groups of starred boxes and X boxes and used CSS layout properties to position and style them.

- **Exercise 4: Selecting Specific Boxes**  
  Practiced selecting specific boxes from a list using `nth-child()` selectors.
